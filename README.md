# 三维道路中心线计算与可视化

一个纯前端的网页应用，用于解析纬地 HintCAD 5.83 数据文件并计算三维道路中心线坐标。

## 功能特性

- **数据导入**：支持解析纬地 HintCAD 的 .pm / .zdm / .sup 文件
- **三维计算**：根据里程桩号计算平面坐标、高程、方位角、坡度等参数
- **可视化展示**：平面线形图、纵断面图
- **里程查询**：查询任意桩号的三维坐标和参数
- **数据导出**：CSV 格式导出

## 使用方法

1. 直接在浏览器中打开 `可视化.html` 文件
2. 上传数据文件或点击「加载示例数据」
3. 点击「解析数据」→「开始计算」
4. 在里程查询中输入桩号获取坐标信息

## 支持的线形

| 类型 | 说明 |
|------|------|
| 直线 | 曲率 k = 0 |
| 圆曲线 | 曲率 k = 常数 |
| 缓和曲线 | 曲率线性变化 |

## 技术栈

- 纯原生 HTML / CSS / JavaScript
- 无外部依赖，单文件运行
- Canvas 2D 绑制图表

## 文件格式

### 平面线元 (.pm)
纬地 HintCAD 导出的平面线元数据，包含线元类型、起终点桩号、坐标、方位角等信息。

### 纵断面 (.zdm)
变坡点数据，包含桩号、高程、竖曲线半径。

### 超高数据 (.sup)
超高渐变段数据，包含左右行车道超高百分比。
